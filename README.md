## Building a Task Management System
## About
Simple **.NET 6** and **C#** web api to manage tasks.

**Entity Framework** was used to connect to a **SQL Server** database on a **Docker** container.

## Features
- Create
- Read all
- Read by id
- Read by title
- Read by date
- Read by status
- Update by id
- Delete by id
